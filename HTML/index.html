Présentation de l'API

L'API TheCocktailDB est une API qui ressence des recettes de cocktail. Il s'agit d'une base de données contenant différentes recettes de cocktails du monde entier.

Explication des différentes requêtes possibles

Pour utiliser l'API, il faut utiliser l'URL suivante : https://www.thecocktaildb.com/api/json/v1/1/search.php?...
